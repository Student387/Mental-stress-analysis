About the System

The Student Mental Stress Analysis System is a Final Year Project designed to support student mental wellness through data-driven assessment. The system evaluates 20 factors including anxiety, sleep quality, academic load, social support, and environmental conditions.

Using a trained machine learning model, the system provides stress level classification (Low, Medium, High) with confidence scores and probability distributions. Results include a detailed analysis of contributing factors and personalized recommendations for improvement.

  • Confidential and anonymous assessment
  • Evidence-based recommendations
  • Resources for meditation, sleep, and support
